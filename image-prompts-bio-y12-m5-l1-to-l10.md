# Image generation prompts — Biology Year 12 Module 5 (Heredity), Lessons 1–10

One generated image per lesson, used in each lesson's hook scene. Every image MUST be a
**transparent PNG with a real alpha channel** (no baked-in background, no white box) and must
contain **no text, letters, numbers or labels** of any kind — all captions are added by the
renderer. Target a clean, modern, flat-but-premium science-illustration look that reads at
1920×1080 over a dark slide. Subject-consistent palette: cool teals, indigos and greens with a
single warm accent; soft cell-shading, gentle rim light, no photographic realism.

Save each file to the exact path given. The camelCase asset key is the value already referenced
in the lesson JSON's hook `image` field (asset registration in `src/assets/index.ts` is the
maintainer's job).
