# ChatGPT image-creator brief — Chem Y11 M1, Lessons 1–10

One concept image per asset below. Each prompt is DIDACTIC — a concrete
scene a muted-volume student can read, not abstract shapes. Paste one at a
time into the ChatGPT image creator and save at the listed path. The asset
registry and lesson JSONs must be wired to these camelCase filenames.

> **Only 4 images are needed for M1 L1–L10.** Every other visual in these
> lessons uses a CODED Remotion diagram (`flow`, `table`, `beforeAfter`,
> `barChart`) — no image required for those scenes.

**Hard rules for every image:**
- 2048 × 2048 PNG, true alpha-channel transparent background (no baked
  colour, no white, no checkerboard)
- NO text, letters, numerals, formulas, or labels anywhere
- Concrete lab apparatus / real chemistry, not abstract geometric shapes
- Chunky isometric 3D, matte, chem-green / amber / charcoal palette
- Match the reference: `public/assets/carbon-12-molar-mass-balance.png`
- End each prompt with: "The file must have true alpha-channel transparency
  so it composites cleanly onto a light slide background."
