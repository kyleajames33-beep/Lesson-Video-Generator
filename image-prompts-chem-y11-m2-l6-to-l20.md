# ChatGPT image-creator brief — Chem Y11 M2, Lessons 6–20

One concept image per lesson. Each prompt is DIDACTIC — a concrete lab
scene a muted-volume student can read, not abstract shapes. Paste one at
a time into the ChatGPT image creator and save at the listed path. The
asset registry and lesson JSONs are already wired to these filenames.

> **UPDATE:** L6, L10, L13, L17, L18 now use CODED Remotion diagrams
> (concentrationCompare, titrationSetup, limitingExcess, titrationSetup,
> errorDartboard) — **no image needed for those five.** Only generate the
> 10 images still listed below: L7, L8, L9, L11, L12, L14, L15, L16, L19, L20.

**Hard rules for every image:**
- 2048 × 2048 PNG, true alpha-channel transparent background (no baked
  colour, no white, no checkerboard)
- NO text, letters, numerals, formulas, or labels anywhere
- Concrete lab apparatus / real chemistry, not abstract geometric shapes
- Chunky isometric 3D, matte, chem-green / amber / charcoal palette
- Match the reference: `public/assets/carbon-12-molar-mass-balance.png`
- End each prompt with: "The file must have true alpha-channel transparency
  so it composites cleanly onto a light slide background."
