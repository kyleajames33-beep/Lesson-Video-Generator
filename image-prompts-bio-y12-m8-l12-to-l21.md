# Image generation prompts — Biology Year 12, Module 8 (Non-infectious Disease and Disorders), Lessons 12–21

One generated image per lesson. Each is a single hook visual.

**Global requirements for every image:**
- Output a **transparent PNG with a real alpha channel** (no background fill, no checkerboard, no white card behind the subject).
- **No baked-in text, labels, numbers, captions, watermarks or logos** anywhere in the image (all on-screen text is added by Remotion).
- Clean, modern, didactic editorial-illustration style; flat vector / soft-shaded look; confident line work; restrained palette built around the Biology blue (#1f6fb2 / #3a8ad9) with a single warm accent (amber #f0a830) reserved for the one element that should draw the eye.
- Subject centred with generous transparent margins; balanced for a 1920×1080 frame with the subject occupying roughly the left or right two-thirds (composition can breathe; on-screen text sits opposite).
- Scientifically accurate proportions; no surreal anatomy; no gore.
