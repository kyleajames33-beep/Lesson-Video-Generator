# Image generation prompts — HSC Biology Year 12, Module 8 (L1–L11)

One generated image per lesson (the hook visual). Every other visual in these lessons
uses existing coded diagrams (`flow`, `table`, `barChart`, `lineGraph`) and needs no image.

## Global requirements (apply to EVERY prompt)
- **Transparent PNG with a real alpha channel** (not white, not a checkerboard baked in).
- **No baked-in text, labels, numbers, captions, watermarks, or UI** of any kind — all text is added in Remotion.
- Aspect: square-ish framing, subject centred with generous padding so it floats on the dark slide.
- Style: clean, modern editorial science illustration; soft teal/sky/mint palette to match the Biology "Coastal" system (teal `#0d7377`, mid `#14a9b8`, mint `#a8e6cf`, sky `#80cfe0`, peach accent `#f5c6a0`); subtle doodled/hand-annotated linework; gentle shadows; flat-but-rich, not photorealistic, not clip-art.
- Single hero subject per image; avoid busy backgrounds (background must be fully transparent).
- Scientifically accurate; no decorative inaccuracies.
